About the Project

This is a movie recommendation system powered by Artificial Intelligence.

How to use?
For each genre, enter a value from 0 (you don't like it) to 1 (you love it) to indicate how much you enjoy that genre. For example, if you like Action a lot, put 1; if you don't like Horror, put 0; and you can use values like 0.5 if you are neutral.

After filling your preferences, click Get Recommendation and the AI will suggest a movie that best matches your taste, using machine learning techniques.

How does it work?
- The model was trained using a simple dataset of movies and user preferences found on the internet, mainly based on genres. It uses Python, Flask, and basic machine learning libraries.

Stack: Python, Flask, AI, HTML, CSS.
